## Documentation Standards

All documentation follows these principles:

1. **Actionable** - Clear next steps for the reader
2. **Concise** - Essential information only
3. **Organized** - Logical flow with clear sections
4. **Current** - Updated with each major change
5. **Referenced** - Links to related documentation
